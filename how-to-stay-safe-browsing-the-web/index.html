How to Stay Safe Browsing the Web

21 Apr, 2025

This is a page about »How to stay safe while browsing the Web«.

Starting out with this post lacking a few things. I will update it later where I also include links to the things I’m talking about.

  1. Install an adblocker on your browser (This can be tricky if you use google chrome, and I highly recommend using another browser.)

Why install an adblocker?

I have experienced so many times where I’ve told a friend or family member to download a program from a website I’ve specified. They click download on the website I’ve specified, and to my surprise they didn’t download the correct thing. The reason being, I use an adblocker but they didn’t and the ads they see say “download”. This clearly shows that ads are a huge security nightmare. There’s also so many other annoyances ads include, sponsored search results (ads) can be leading you to a website containing malware, they can play unwanted sound and the list goes on. So really, I very much recommend you to Install an adblocker.

  1. Install a browser extension like Malwarebytes Browser Guard

The best security against malicious actors is yourself.
